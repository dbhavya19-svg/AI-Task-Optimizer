# 🤖 AI-Powered Task Optimizer

## 📌 Overview
The **AI-Powered Task Optimizer** is a data science-based application that analyzes employee emotions using text input and recommends suitable tasks to improve productivity and well-being.

---

## 🎯 Features
- 🧠 **Sentiment Analysis** (Happy, Sad/Stressed, Neutral)
- 📋 **Task Recommendation System**
- ⚠️ **Stress Detection Alerts**

---

## 🛠️ Technologies Used
- Python
- Streamlit
- TextBlob (NLP)
